# 🏆 FIFA Qatar World Cup 2022 — Power BI Dashboard

## Overview
An interactive Power BI dashboard analyzing match statistics from the FIFA World Cup Qatar 2022. Built to compare team performances, track discipline trends, and explore tournament-wide patterns through dynamic visuals and DAX measures.


---

## Features & Insights

**Match Performance**
- Possession rate comparison between **home** and **away** teams
- Total and on-target shot attempts per match
- Yellow and red card breakdown by team
- Penalties awarded and offside calls per game

**Tournament Analysis**
- Team-wise performance comparison across all matches
- Match attendance figures per stadium
- Referee details per fixture
- Home vs away performance trends throughout the tournament

**Interactivity**
- Filter by match category, home team and away team
- Dynamic slicers for real-time data exploration
- KPI indicators for quick summary metrics
